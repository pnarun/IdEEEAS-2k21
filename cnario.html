"C"Nario

A thrilling programming contest with a big prize at stake will be up for grabs this year. The language used is ‘C’ as it is the most commonly used and taught language. The students will be tested in their analytical and logic skills. The event is divided into two rounds namely, MCQ round, and a final coding round.

  • Event starts at 11:45 AM on 26th June.

  • Entry Fee: Rs.50/- per participant.

  • 20% Discount will be provided to IEEE Membership holders.

  • All registered candidates will get the confirmation mail.

  • Registration closes on 25th June at 12:00 AM.

  • Participation e-certificate will be provided.

  • One can participate in any number of events. There won't be a time clash between events.

Rules:

  • This is a programming contest purely based on C language.
  • Video proctoring and plagarism check will be done.
  • Judges decision is final.
  • The platform used is CODING NINJAS.

    • Round 1(MCQs)

  • Each member will be given 30 MCQs.
  • Time duration is 30 mins.
  • Shortlisted participants will be called for the Coding round.

    • Round 2(Coding)

  • This is a program building round of 90 mins.
  • It consists of 3 program statements which should be solved using C language.

    • Judging Criteria:

  • The participant finishing all the programs in least time will be announced as the winner.
